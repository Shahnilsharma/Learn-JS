API stands for Application Programming Interface. It is a set of rules and protocols that allows different software applications to communicate and interact with each other. APIs define the methods, data formats, and conventions that developers can use helpful for retrieving data from a remote server, sending data to a third-party service, integrating different software components, or even controlling hardware device earlier api were requested usin ajax(Asynchronous Javascript and xml) method there are 5 states 0: request not initialized 1: server connection established 2: request received 3: processing request 4: request finished and response is ready
